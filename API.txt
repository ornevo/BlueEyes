- WEBSOCKET based
- fetching words
- adding a word - severity + word
- server sends a push each time the notifications are updated
- mark notification as done
- fetch text and audio of a notification
- submit audio sample

format of a notification:
{
    id: "whatever",
    severity: 1/2/3,
    words: ["id of words"],
    freq: 52.5,
    time: "11:24",
    date: '1.2.2020',
    audio: 'http://link-to-mp3-sample.mp3',
    text: 'the STT text in the audio sample'
}

format of a word:
{
    id: "whatever",
    word: "תירס חם",
    severity: 1/2/3
}
